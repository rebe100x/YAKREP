CloudView SDK PHP API Clients - Beta
====================================

In this package you will find all the components that you need to build applications on top of the Exalead CloudView platform using the PHP programming language.

In addition, samples using the SDK are provided.

Ths package contains the following directory and files:

* papi/ {containing the PAPI client library}
- README.txt
* search/ {containing the Seach client library}


Global requirements:
===================

In order to develop an application using the CloudView SDK PHP Clients
libs in your project environment, you must add all the SDK libraries
(listed above) to your project.

 * CloudView edition version 5.1.X
 * PHP 4 or higher
